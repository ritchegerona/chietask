Install & run (full multi-user SaaS)

The browser demo needs no install. Use these steps when you want accounts, workspaces, and the API on your machine.

1. Requirements

DependencyNotes
Python 3.10+Required
pipComes with Python
GitTo clone the repo
Modern browserChrome, Firefox, Safari, Edge
Docker (optional)Alternative to local Python

2. Python packages

From backend/requirements.txt:

  • Runtime: fastapi, uvicorn, sqlalchemy, python-jose, bcrypt, python-multipart, email-validator, pydantic, pydantic-settings, python-dotenv
  • Tests: httpx, pytest

3. Clone & install

git clone https://github.com/ritchegerona/chietask.git
cd chietask
python3 -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate
pip install --upgrade pip
pip install -r backend/requirements.txt

Or: make install

4. Run

python run.py
# or: make run
# open http://localhost:8765

macOS: double-click start.command.

5. Docker

docker compose up --build
# http://localhost:8765

6. Tests

pytest backend/tests -q
# or: make test

7. First use

  1. Open http://localhost:8765/register
  2. Create an account (workspace is created automatically)
  3. Add tasks, categories, timers
